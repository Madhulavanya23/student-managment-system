# 🎓 Student Management System

A simple Java-based application to manage student records using CRUD operations.

## 🚀 Features

* Add student details
* View all students
* Update student information
* Delete student records

## 🛠️ Tech Stack

* Java
* Object-Oriented Programming (OOP)

## ▶️ How to Run

```bash
javac Main.java
java Main
```
